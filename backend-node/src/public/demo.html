AvaChat Live Chat Widget

Klik tombol chat di kanan bawah untuk memulai percakapan.

Widget Features

  • Embeddable JavaScript snippet (no dependencies)
  • Pre-chat form (nama customer)
  • Real-time messaging via Webhook Simulator
  • File attachment support (image, PDF, DOC)
  • File size limit: 10MB
  • Auto-sentiment analysis incoming messages
  • FAQ auto-reply detection
  • Responsive design (mobile fullscreen)
  • Unread message badge

Embed Code

<script src="http://localhost:4000/widget/widget.js" async></script>
